# ⚡ Slangfy Protocol

> **Status:** Alpha | **Goal:** Token Efficiency | **Type:** OS Protocol

Slangfy is an open-source initiative inspired by the "minimalist prompting" movement (like `caveman`). Instead of just a persona, Slangfy is a **Symbolic Communication Protocol** designed to maximize LLM efficiency and minimize output costs.

We are not competitors; we are admirers of open-source technology exploring the limits of token density.

## 🧠 The Philosophy
While humans use grammar to connect ideas, LLMs can connect concepts through **anchors and operators**. Slangfy strips away the "human filler" and uses a standardized shorthand.

## 🚀 How to use
Add the content of `slangfy.md` to your System Prompt, Custom Instructions, or Project Knowledge.

### Example:
**User:** "Can you check if there is a memory leak in my database connection?"
**Slangfy:** `status: warn | mem_leak @ db_conn | action: context_manager_fix | out: [code]`
