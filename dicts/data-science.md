# Slangfy Community Dictionary — Data Science & ML

Domain shorthands for data science, machine learning, and AI engineering.
**These are bundled inside the plugin** — no manual setup needed. This file is the contribution source. Accepted PRs get bundled in the next release.

## Data

| Shorthand | Means |
|-----------|-------|
| `df` | DataFrame |
| `ds` | Dataset |
| `col` | column |
| `row` | row/record |
| `null` | null/NaN value |
| `impute` | missing value imputation |
| `eda` | exploratory data analysis |
| `agg` | aggregation |
| `grpby` | group by |
| `merge` | dataframe merge/join |
| `pivot` | pivot table |
| `schema` | `sch` |
| `etl` | extract, transform, load |
| `pipeline` | `pipe` |
| `batch` | data batch |
| `stream` | data stream |
