# Slangfy Community Dictionary — Web Dev

Domain shorthands for frontend and backend web development.
**These are bundled inside the plugin** — no manual setup needed. This file is the contribution source. Accepted PRs get bundled in the next release.

## Frontend

| Shorthand | Means |
|-----------|-------|
| `DOM` | Document Object Model |
| `props` | component properties |
| `hook` | React/Vue hook |
| `store` | state store (Redux, Pinia, Zustand) |
| `emit` | event emitter |
| `slot` | template slot |
| `SSR` | server-side rendering |
| `CSR` | client-side rendering |
| `SPA` | single-page application |
| `PWA` | progressive web app |
| `HMR` | hot module replacement |
| `vdom` | virtual DOM |
| `ctx` | context |
| `ref` | reactive reference |
| `computed` | computed property |
| `watch` | reactive watcher |
| `layout` | page layout component |
| `route` | router route |
| `guard` | navigation guard |
| `hydrate` | hydration (SSR) |
| `bundle` | compiled output bundle |
| `chunk` | code-split chunk |
| `tree-shake` | dead code elimination |
| `a11y` | accessibility |
| `i18n` | internationalization |
| `l10n` | localization |
