# Slangfy Community Dictionary — DevOps & Infrastructure

Domain shorthands for DevOps, cloud, and infrastructure engineering.
**These are bundled inside the plugin** — no manual setup needed. This file is the contribution source. Accepted PRs get bundled in the next release.

## CI/CD

| Shorthand | Means |
|-----------|-------|
| `CI` | continuous integration |
| `CD` | continuous deployment/delivery |
| `pipe` | CI/CD pipeline |
| `stage` | pipeline stage |
| `job` | pipeline job |
| `runner` | CI runner/agent |
| `artifact` | build artifact |
| `registry` | container/package registry |
| `tag` | image/release tag |
| `release` | release version |
| `rollback` | deployment rollback |
| `canary` | canary deployment |
| `blue-green` | blue-green deployment |
